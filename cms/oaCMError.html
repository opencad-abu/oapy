oaCMError

模块: oapy._oa._cms 导入: from oapy._oa import _cms

概览

oaCMError 在 oapy 中可用，可通过 _cms 模块访问。

本页汇总 oaCMError 当前在 oapy 中可用的 Python 接口。

详细说明

The oaCMError class is a concrete class that is used to throw exceptions in the OpenAccess change management system (CMS) 。 The oaCMError class supports a variable number of arguments。

构造函数

_cms.oaCMError(msgIdIn, ...)
