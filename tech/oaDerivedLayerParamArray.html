oaDerivedLayerParamArray

模块: oapy._oa._tech 导入: from oapy._oa import _tech

概览

oaDerivedLayerParamArray 在 oapy 中可用，可通过 _tech 模块访问。

本页汇总 oaDerivedLayerParamArray 当前在 oapy 中可用的 Python 接口。

详细说明

This class represents an array of oaDerivedLayer parameters。

构造函数

_tech.oaDerivedLayerParamArray(sizeIn=0)

obj = _tech.oaDerivedLayerParamArray(sizeIn=0)

方法总览

状态 Python 调用
✅ obj.findParam(def)

方法说明

obj.findParam(def)

绑定状态: 已绑定

Python 调用: obj.findParam(def)

This function searches the derived layer param array for a parameter with the specified definition。 If the definition is found, the parameter is returned, otherwise NULL is returned。

参数

  • def: The parameter definition of the derived layer parameter to return。

Python 示例

from oapy._oa import _tech

# assume obj is a oaDerivedLayerParamArray
obj.findParam(def)

方法

  • _tech.oaDerivedLayerParamArray(sizeIn=0)
