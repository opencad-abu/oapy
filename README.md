# oapy

Si2 OpenAccess 22.61 Python bindings via pybind11.

## Install

```bash
# 当前系统 (glibc 2.34+)
pip install dist/oapy-0.1b0-cp312-cp312-linux_x86_64.whl

# RHEL7/CentOS7 (glibc 2.17+)
pip install dist/oapy-0.1b0-cp312-cp312-glibc-2.17_x86_64.whl
```
